Esta Página

Arquitectura

Esta página está montada sobre un servidor HTTP corriendo sobre Node.JS, en los servidores de Heroku, cuyo código y despliege fue realizado enteramente por mi. En el back-end usé ExpressJS como framework para montar un servidor HTTP sirviendo contenido estático, no require base de datos. Por otro lado, en el front-end usé Bootstrap 4 y tomé ideas prestadas de otras páginas de desarrolladores y plantillas de Bootstrap, aunque la totalidad del código en JavaScript, CSS y HTML lo escribí 'a mano'.
